# Tarea-5JS-2026
### Ejercicios de fibonacci y arrays de pokemon
- Ejercicio 1: mostrar secuencia de fibonacci con metodo for y una funcion para que me muestre los numeros segun calculo de fibonacci. Ademas de crear una funcion para guardar esos números en un arreglo y poder usarlos despues en los ejercicios b y c.
- Ejercicio b del 1 se realizo con método .for each para que me trajera del array todos los numeros pares, que fueran divisibles entre 2 con resultado igual a 0.
- Ejercicio c del 1 Se realizo igual con método .for each para que me trajera del array todos los numeros impares, o que no fueran divisibles entre 2 con resultado diferente a 0.
- Ejercicio 2: Se utiliza el método .map y el .toUpperCase para que recorriera el array y devolviera un array nuevo con los mismos elementos en mayusculas.
- Ejercicio 3: Del array dado de pokemones se debia filtrar los tipo fuego, utilice el método .filter que me trajo solo los tipo fuego.
